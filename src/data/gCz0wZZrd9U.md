
## Nova Lite
### 대규모 모델 학습 과정에서의 도전과 AWS의 솔루션
* 대규모 모델 학습은 최신 하드웨어 사용, 오류 대응, 예측 가능한 타임라인, 성능 최적화, 비용 관리 등의 도전을 필요로 한다.
* AWS의 SageMaker HyperPod는 이러한 도전을 해결하기 위해 도입되었으며, 학습 시간을 최대 40%까지 단축할 수 있다.
* HyperPod는 학습 중 발생하는 오류를 자동으로 대응하고, 클러스터 내에서 모델과 데이터를 효율적으로 분산하여 학습 성능을 최적화한다.

### 학습 계획 및 레시피를 통한 학습 효율 향상
* 학습 계획은 EC2 용량 블록을 활용하여 학습에 필요한 용량을 미리 예약할 수 있게 해주며, 예약된 용량을 활용하여 학습 클러스터를 자동으로 설정한다.
* 학습 레시피는 대규모 모델의 미세 조정 및 사전 학습을 위한 준비된 레시피를 제공하여, 데이터 사이언티스트와 머신러닝 엔지니어가 빠르게 학습을 시작할 수 있도록 돕는다.
* 레시피는 여러 가지 설정을 미리 구성하여 학습 성능을 최적화하고, 자동 체크포인트 기능을 통해 학습 중 발생하는 오류를 효과적으로 대응할 수 있다.

### 실제 케이스: NinjaTech AI의 혁신적 학습 방법
* NinjaTech AI는 대규모 모델 학습을 위해 HyperPod 레시피를 활용하여 학습 과정을 크게 간소화하고 효율성을 향상시켰다.
* 대규모 모델의 미세 조정과 사전 학습을 통해 고객의 요구에 맞춘 고성능 모델을 제공하고, 학습 비용을 절감하였다.
* 특히, 대규모 모델의 대화형 학습을 통해 새로운 기능을 개발하고, 고객에게 더 나은 서비스를 제공하는 데 성공하였다.

### AWS SageMaker HyperPod의 트레이닝 플랜 소개
* 대규모 모델 학습에서 최신 하드웨어 활용, 장애 처리, 일정 관리, 성능 최적화, 비용 관리 등의 다양한 과제가 있다.
* 트레이닝 플랜은 EC2 용량 블록을 활용하여 컴퓨팅 자원 확보의 불확실성을 제거하고 수동 프로세스를 최소화한다.
* 사용자는 인스턴스 유형, 수량, 학습 기간, 시작 시점 등을 지정하여 필요한 컴퓨팅 자원을 예약할 수 있다.
* 트레이닝 플랜은 자동으로 학습 클러스터를 설정하고 예약된 시점에 컴퓨팅 자원을 활성화한다.

### HyperPod 레시피 기능 소개
* 레시피는 Llama, Minstral, Mixtral 등 인기 있는 공개 기반 모델의 사전 학습과 미세조정을 위한 최적화된 구성을 제공한다.
* 레시피를 통해 몇 주가 걸리던 학습 준비 과정을 몇 분으로 단축할 수 있다.
* 자동 체크포인팅, 효율적인 자원 활용, 비용 최적화 등의 기능을 제공한다.
* 시퀀스 길이, 모델 크기 등을 쉽게 조정할 수 있고 GPU나 Trainium 등 다양한 가속기 사용이 가능하다.

### NinjaTech AI의 활용 사례
* NinjaTech AI는 HyperPod 레시피를 활용하여 Llama 모델에 음성 대화 기능을 추가하는 학습을 수행했다.
* 다중 노드 구성으로 14시간 만에 학습을 완료했으며, 비용은 5,000달러 미만이었다.
* Super Agent 기술을 통해 여러 AI 모델을 결합하여 성능을 향상시켰으며, Arena Heart 벤치마크에서 최고 성능을 달성했다.
* HyperPod 레시피를 통해 복잡한 학습 환경 설정이 간소화되고, 용량 예약과 비용 절감이 가능해졌다.
